## Customization

All colors, text, and content can be customized through the Shopify theme editor. Navigate to **Online Store > Themes > Customize** in your Shopify admin.

## Installation

1. Zip the theme folder
2. Go to Shopify Admin > Online Store > Themes
3. Click "Upload theme" and select the zip file
4. Customize as needed!
